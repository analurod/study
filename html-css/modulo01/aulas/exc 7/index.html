Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b> (não semântica)

Nesta frase temos um termo em negrito usando a tag <strong> (semântica)

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag <i> (não semântica)

Nesta frase temos um termo em itálico usando a tag <em> (semântica)

Texto marcado

Podemos criar também um texto marcado usando a tag <mark>

Para editarmos a cor de todas as marcas da página de uma vez utilizamos o style no cabeçalho, agora para uma única tag adicionarmos o style na mesma

Textos grandes e pequenos

Estamos criando um texto grande e um texto pequeno utilizando as tags <big> e <small> respectivamente.

Porém a tag big está obsoleta, já que não tem sentido para estar ali já que podemos formatar.

Textos deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido mas não considerado, utilizamos a tag <del>

Texto inserido

Podemos marcar um texto como inserido para dar Ênfase que ele foi adicionado depois, utilizamos a tag <ins> (semântica)

Podemos também usar a tag <u> mas a mesma está obsoleta (não semântica)

Texto sobrescrito e subscrito

Sobrescrito

Para escrever coisas do tipo x2+35, utilizamos a tag <sup>

Subscrito

Para escrever coisas do tipo H2O, utilizamos a tag <sub>
