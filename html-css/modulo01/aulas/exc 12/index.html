Estilo Inline

No estilo inline colocamos o parâmetro de estilo (style) em cada tag, o que é um fator limitante e deixa o código poluído, é a forma menos eficiente.

Abaixo temos um texto que exemplifica as alterções de estilo, todas foram feitas em uma tag por vez, como por exemplo <h2 style="color: blue;">Capítulo 1</h2> ou <body style="background-color: aquamarine;">
