Citaçãoes e Códigos

Código fonte

O comando document.getElementById('teste') é escrito em JavaScript e para utilizar essa formatação diferenciada para códigos utilizamos a tag <code>. Veja abaixo outro exemplo:

        
    num = int(input('Digite um número'))
    if num % 2 == 0
        print(f'O número é par!')
    else
        print(f'O número é impar!')

Para incluir a identação do código utilizamos a tag <pre>.
