Imagens

Não podemos pegar qualquer imagem da internet e colocar no nosso site, devido os direitos autorais. Para isso devemos ficar atentos quanto os diretos de imagem.

Tem alguns sites como Unsplash, Pexels e Freepik que podemos encontrar imagens que PODEM ser utilizadas.

Para editarmos imagens podemos utilizar o Canva ou GIMP. Temos também o Photoshop, porém o mesmo é pago.


Principais formatos de imagem para web:

1. JPEG - Criado em 1983, consegue compactar imagens MUITO BEM
2. PNG - Criado em 1996, criado para substituir o GIF, apesar disso não possui animação.Permite fundo transparente e tem maior qualidade


O tamanho das imagens importa para construções de sites? SIM!

1500px é uma boa largura, já resolução podemos colocar algo entre 50 e 75.
Em sites normalmente utilizamos 650px em largura.
Devemos nos atentar quanto a imagem devido o tamanho do arquivo.

Podemos redefinir o tamnho da imagem com CSS, porém o espaço não diminui, ou seja, teriamos uma imagem menor mas que ocupa a mesma memória. Isso pode deixar o site mais lento e sites mais lentos não tem destaque no Google!
