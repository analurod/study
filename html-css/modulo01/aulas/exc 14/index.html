Estilos Externos

Adicionar os estilos externos é a forma mais versátil de adicionar estilo CSS ao seu conteúdo. Para isso utilizamos a tag <link rel="stylesheet" href="style.css"> no head do arquivo HTML.

Podemos usar mais de um link de estilo. Além disso, podemos usar mais de um estilo na página, a ordem de "prioridade" é Inline, Local e Externo respectivamente.
