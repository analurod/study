Imagens Dinâmicas

Resolução e Tamanhos

O usúario vai acessar seu site em dispositivos diferentes (celular, tablet,computador) portanto é importante se atentar a resolução e tamanho da imagem em diferentes dispositivos.

Tente abrir este site em vários dispositivos diferentes ou aumente e diminua o tamanho do navegador. Para adicionar a imagem dinâmica ao site utilizamos o código abaixo:

        
            <picture>
                <source media="(max-width: 780px)" srcset="imagens/pequena.png" type="image/png">
                <source media="(max-width: 1080px)" srcset="imagens/media.png" type="image/png">
                <img src="imagens/grande.png" alt="Imagem fléxivel">
            </picture>

Para que a imagem se adapte ao tamanho da tela usamos a tag <picture> e <source>, assim podemos definir a partir de qual tamanho devemos exibir qual resolução de imagem.

Áudio

Podemos colocar áudios em nossos sites apenas utilizando HTML5. São compatíveis os formatos MP3, WAV e OGG, é bom termos o arquivo em diferentes formatos para melhora a experiÊncia do usuário. Lembrando os áudios também tem direitos autorais.

Formas de Adicionar Áudio

No primeiro caso utilizamos apenas a tag <audio> para adicionar o áudio a página e inicar a reprodução automática do mesmo, como mostra abaixo:

        
            <audio src="midias/happy-mistake.mp3" controls autoplay></audio>

Já nesse segundo caso, utilizamos a tag <audio> e a tag <source:src>, como mostrado abaixo:

        
            <audio preload="metadata" controls autoplay>
                <source src="midias/guanacast-33.mp3" type="audio/mpeg">
            </audio>

A primeira maneira é a mais simples porém mais limitada em questão de formatos, então para maior compatibilidade utilizamos a segunda forma.

Parâmetros adicionais nas Tags <audio>

Controls

O "controls" dentro da tag <audio> serve para que possamos ver essa barra de controle do áudio.

Autoplay

O "autoplay" dentro da tag <audio> serve para iniciar automaticamente o áudio.

Preload

Não colocar nada na tag <audio> ou colocar preload auto faz com que o carregamento do site só seja considerado completo assim que o site terminar de carregar o áudio todo, é perigoso para arquivos grandes.

Com o preload metadata vai ser carregado alguns dados só.

Já com preload none só vai ser carregado quando o usuário apertar o play.

Loop

Para o áudio ser reproduzido novamente após chegar ao fim basta colocarmos o termo "loop" na tag <audio>

Vídeos

Os formatos compatíveis mais comuns são MP4, M4V, WebM e OGV, assim como no áudio é importante termos o vídeo em diferentes formatos para garantir uma experiência boa a maioria dos usuários.

Para converter os vídeos podemos utilizar o "HandBrake" de forma gratuita.

Hospedagem Própria

Abaixo temos vídeos hospedados no próprio servidor. Assim como aúdio podemos inserir da forma mais simples, mas como queremos maior compatibilidade iremos inserir da forma a seguir:

        
            <video width="500" poster="imagens/capa-video.png" controls autoplay loop>
                <source src="midias/video-editado-web.mkv" type="video/webm">
                <source src="midias/video-editado-mp4.mp4" type="video/mp4">
                <source src="midias/video-editado-ogv.ogv" type="video/ogg">
                <p>Seu navegador não tem compatibilidade com reprodução de vídeos. Clique aqui para fazer o download.<p>
            </video>

O parâmetro poster adiciona uma capa ao vídeo.
